# Personal Portfolio — Valeriia Barabash

A modern, responsive personal portfolio website built to present my skills, projects, and professional background.  
The design focuses on clarity, accessibility, and smooth user experience across all devices.

## 🚀 Features

- **Responsive layout** — fully optimized for desktop, tablet, and mobile  
- **Clean navigation** with icon‑based menu  
- **Hero section** introducing my role and personality  
- **Project slider** showcasing my recent work  
- **Education timeline** with animated elements  
- **Contact form** connected via FormSubmit  
- **Modern UI** using soft shadows, pastel colors, and smooth animations  

## 🛠️ Technologies Used

- **HTML5**  
- **CSS3** (custom responsive design, animations, timeline effects)  
- **JavaScript** (slider initialization, interactions)  
- **Splide.js** for project carousel  
- **FormSubmit** for contact form handling  

## 📂 Sections Overview

### 🔹 Home  
Intro card with my photo, contact details, and a short description.

### 🔹 Works  
A slider with previews of my projects (Cafe website, Creative Agency, Travel Agency).

### 🔹 Education  
A vertical animated timeline showing my academic and professional path.

### 🔹 Contact  
A simple and functional form for sending messages directly to my email.

## 📱 Responsive Design

The website includes custom media queries for:  
- 1024px (tablets)  
- 768px (mobile landscape)  
- 480px (small mobile screens)

The navigation transforms into a **mobile bottom bar** for better usability.
